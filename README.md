# sewei-personal-skill

我的 skill 仓库。

Personal Codex skills repository for storing, versioning, and sharing reusable skills.

## Structure

```text
skills/
  skill-name/
    SKILL.md
    agents/
      openai.yaml
    references/
    scripts/
    assets/
```

## Usage

Copy a skill folder into `skills/`, commit the change, and push it to GitHub.

To install one of these skills into Codex later, use the GitHub path for that skill folder.
